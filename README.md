# design-002
스토어 서비스(금강상사)

## 리소드

[부트스트랩](https://getbootstrap.kr/)

적용버전

```sh
# css
https://cdn.jsdelivr.net/npm/bootstrap@5.2.2/dist/css/bootstrap.min.css
# js
https://cdn.jsdelivr.net/npm/bootstrap@5.2.2/dist/js/bootstrap.bundle.min.js
```

### 폴더구조

```sh
root
    /str # 고객용(store)
    /wrk # 직원용(work)
    /adm # 관리자(admin)

    /img # 이미지
    /libs # 웹 리소스, 라이브러리
        /bootstrap # 부트스트랩
```
