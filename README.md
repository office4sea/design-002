# design-002
스토어 서비스(금강상사)

## 리소스

[부트스트랩](https://getbootstrap.kr/)

적용버전(5.2.2)

### 폴더구조

```sh
root
    /hpg # 고객용(homepage)
        # 구글 플레이 스토어 구조 참고
        # https://play.google.com/store/games
    /wrk # 직원용(work)
    /adm # 관리자(admin)

    /img # 이미지
    /libs # 웹 리소스, 라이브러리
        /bootstrap # 부트스트랩
```
